## Country per Alpha2 Code
### Pais por Codigo Alpha2

Obten el nombre de un pais por su codigo alpha2.
Por ejemplo: MX = 'Mexico', US = 'United states', TR = 'Turkey', BG = 'Bulgaria'.
Espero y les guste :).
